# Compilar contratos
npx hardhat compile

# Rodar rede local
npx hardhat node

# Rodar script de deploy
npx hardhat run scripts/deploy/deploy.js --network mumbai                  ** REDE MUMBAI **
npx hardhat run scripts/deploy/deploy.js --network localhost               ** REDE LOCAL  **

# Executar testes
npx hardhat test **xXxXxXxXxXx**

# Executar console painel hardhat
npx hardhat console --network localhost                             ** REDE LOCAL  **
npx hardhat console --network munbai                                ** REDE MUMBAI **



Recursos de Nível Empresarial
Otimização Avançada:
Compilador configurado com optimizer.runs: 1000, viaIR: true, e yul: true para bytecode altamente otimizado.
Suporte a múltiplas versões do Solidity (0.5.16, 0.6.6, 0.8.20, 0.8.28) para Uniswap V2, V3 e V4.
Suporte a Múltiplas Redes:
Inclui localhost, Mumbai, Polygon, Ethereum Mainnet, Sepolia, Arbitrum Sepolia e Optimism Sepolia.
Configurações de timeout (120 segundos) e gas otimizadas para estabilidade em redes instáveis.
Integração com Ferramentas:
hardhat-verify: Verificação automática em Etherscan, Polygonscan, Arbiscan e Optimism Explorer.
hardhat-abi-exporter: Gera ABIs organizados para integração com front-end.
hardhat-deploy: Gerencia deploys reutilizáveis com namedAccounts.
hardhat-tracer: Depuração detalhada com logs estruturados e custos de gás.
hardhat-storage-layout: Gera layouts de armazenamento para contratos atualizáveis.
hardhat-docgen: Documentação automática em Markdown para contratos.
@tenderly/hardhat-tenderly: Monitoramento e simulação de transações via Tenderly.
Testes em TypeScript:
Suporte para arquivos .spec.ts via ts-node/register.
Relatórios de teste em HTML/JSON com mochawesome.
Relatórios e Monitoramento:
hardhat-gas-reporter: Relatórios detalhados de gás em USD, excluindo mocks.
hardhat-contract-sizer: Monitora tamanhos de contratos com foco em Uniswap.
Typechain:
Gera tipagens TypeScript para contratos com ethers-v6, incluindo artefatos externos.
CI/CD e Segurança:
Configurações prontas para integração com pipelines CI/CD (ex.: GitHub Actions).
Suporte para verificação privada via Tenderly.
Layouts de armazenamento para auditorias de contratos atualizáveis.
